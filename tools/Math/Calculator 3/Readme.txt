————————————软件介绍————————————
Mathemetics Calculator 3可以用 set 命令自定义变量，也可以用 view 今天查看变量的值。
在程序里有一个设置小数精度的全局变量 prec，可以用 set 对它重新赋值，值的范围是0－100的整数，默认是 2 。
将sin、arccot、abs、ln等数学函数转换成内部表示，以便于将之与一般的运算符统一处理。
Calculator还对小数和整数加以了区别，新增求余去处符 ‘\’、百分符‘%’

Calculator主要采用了链表、栈等常用数据结构:
链表——用来将表达式字符串格式化为一连串的项和实现对常量、自定义变量的存储；
栈用——以实现程序的核心问题逐一计算。

Calculator主体分为三大部分：输入与命令解释、输出与命令执行 以及 表达式计算。
看起来主体很简约，将计算模块封装成一个没有参数的主体Calculator计算函数

主体Calculator函数从全局变量 expString 获得表达式字符串，进行一系列的分析处理后，得到一个标准表达式(一个表达式链表)，然后进行计算模块。
输入与输出两部分技术还不成熟，命令解释的主要思想先将字符串格式化为至多有三个参数的命令行，与命令列表进行逐一匹配，如果解释成功就利用全局变量产生相应的信息，传递给命令执行函数；命令执行函数根据命令解释的信息决定是否调用计算模块 Calculator函数 控制是否显示结果等等。

Calculato有较为全面的错误报告机制，能对大多数出现的错误进行准确的报告。
主要思想是通过全局错误标志变量 error ，在出现错误的地方将 error 赋予不同值，最后在输出与命令执行部分对错误进行报告，没有错误则变量为NULL(空)。

————————————附加说明————————————
⒈表达式的项
    一个操作数或者一个操作符就是一个项，如 256 、+ 、- 、sin 分别是表达式的四个项。

⒉表达式
    表达式是操作符和操作数加上括号组成的字条序列，其中最外层括号可以省略，其他括号都不能省略，一个表达式有且只有一个可以计算的值。下面是表达式的递归定义:
⑴ 一个操作数是一个表达式，如 1024.64
⑵ 一个一元操作符和一个操作数组成的序列是一个表达式，如 sin32 或 -512.32
⑶ 一个操作数和一个二元操作符和一个操作数组成的序列也是一个表达式，如 1024+2048
⑷ 不省略上述三步的最外层括号(函数操作符除外)，由以上三步组成的有限字条序列是一个表达式。
    特殊说明:
    ① 在表达式中，如果*(乘)两边的操作数是一个符合上述表达式定义的子表达式，则该*可以省略。
    ② 当一个操作数后面是一个符合上述定义的子表达式时，这个操作数作为一个表达式最外层括号可以省略。

⒊运算符的优先级和结合性说明
    下表列出的运算从上到下优先级依次降低，同在一行的优先级相同。

运算符─┤优先级├─结合性

%(百分数)─┤0├─从左往右
数学函数运算符─┤1├─从右往左
^(乘幂)─┤2├─从右往左
-(负)─┤3├─从右往左
*(乘)  /(除)  \(求余数)─┤4├─从左往右
+(加)   -(减)─┤5├─从左往右
()括号─┤6│
